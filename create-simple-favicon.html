Generate Favicon Files

To create proper favicon files from your LLP logo SVG:

  1. Visit RealFaviconGenerator.net
  2. Upload your images/LLP_Logo_Stacked.svg file
  3. Download the generated favicon package
  4. Extract and place the files in your project root

Or use favicon.io converter
